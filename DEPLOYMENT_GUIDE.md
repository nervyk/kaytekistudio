## 🚀 Что вам нужно сделать:

### ЭТАП 1: Создание GitHub репозитория

1. Откройте браузер и перейдите на https://github.com/new
2. Заполните форму:
   - **Repository name**: `kaytekistudio`
   - **Description**: `Portfolio and services website for KAYTEKI Studio`
   - **Public** - выберите
3. ❌ НЕ инициализируйте с README (оставьте всё как есть)
4. Нажмите **"Create repository"**

### ЭТАП 2: Загрузка проекта на GitHub

После создания репозитория вы увидите страницу с инструкциями. Скопируйте и выполните эти команды в PowerShell:

```powershell
cd "d:\KAYTOKI\site"
git remote add origin https://github.com/ВАШ_ЛОГИН/kaytekistudio.git
git branch -M main
git push -u origin main
```

**Замените `ВАШ_ЛОГИН` на ваше имя пользователя GitHub**

При запросе введите ваши учетные данные GitHub.

---

### ЭТАП 3: Развертывание на Netlify

1. Перейдите на https://app.netlify.com
2. Нажмите **"Connect to Git"** или **"New site from Git"**
3. Выберите **GitHub** → авторизуйтесь
4. Найдите и выберите репозиторий `kaytekistudio`
5. В настройках деплоя оставьте всё как есть (пусто в обоих полях)
6. Нажмите **"Deploy site"**

**Netlify выдаст вам ссылку**: `https://XXXXXX.netlify.app`

---

### ЭТАП 4: Добавление кастомного домена

1. В Netlify перейдите в **"Site settings"** → **"Domain management"**
2. Нажмите **"Add custom domain"**
3. Введите: `kaytekistudio.ru`
4. Нажмите **"Continue"** и следуйте инструкциям

**Netlify покажет вам DNS записи для добавления у регистратора:**

Обычно это будут:
- Для `www.kaytekistudio.ru`:
  ```
  CNAME: www.kaytekistudio.ru → XXXXXX.netlify.app
  ```

- Для `kaytekistudio.ru` (@):
  ```
  Либо A records, либо ALIAS/ANAME в зависимости от вашего регистратора
  ```

5. Скопируйте эти DNS записи
6. Зайдите в панель управления вашего регистратора домена
7. Добавьте эти DNS записи в управление доменом
8. Подождите 24-48 часов, пока DNS распространится

4. Выберите **"Open with Live Server"**
